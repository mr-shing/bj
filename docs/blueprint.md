# **App Name**: Behzad Jam Portfolio

## Core Features:

- Homepage Showcase: Interactive 3D showcase of projects, rendered with Three.js, on the landing page, adapting to various screen sizes for optimal viewing.
- Project Details: Dedicated pages with in-depth information on each project, including images, descriptions, and technologies used.
- About Section: A concise 'About Me' section detailing Behzad Jam's background, skills, and professional experience.
- Contact Form: An integrated contact form for inquiries, allowing visitors to send messages directly.

## Style Guidelines:

- Primary color: Deep navy blue (#243A73) to convey professionalism and trustworthiness, while aligning with a modern aesthetic.
- Background color: Light gray (#E5E8EF), offering a clean and unobtrusive backdrop to emphasize the 3D models.
- Accent color: Muted teal (#438A83), used sparingly to draw attention to interactive elements and important information, while complementing the navy.
- Body and headline font: 'Inter', a sans-serif font known for its clean, modern appearance, will be used across the site.
- Minimalist icons for navigation and social links.
- Clean and responsive grid layout ensuring optimal viewing on all devices.
- Subtle animations and transitions for a polished user experience.
